# Asset_Generater Vault

## 목적
2D SRPG용 캐릭터 초상화/케릭터시트/에셋 생성 툴 설계 문서
툴의 개발 목적은 SRPG 개발을 들어가기전에 미리 에셋 생성용

## 현재 목표
- 모듈형 캐릭터 생성 구조 정리
- 헤어/얼굴/투구/색상 시스템 확정
- 데이터 스키마 초안 설계

- 2D 전용
