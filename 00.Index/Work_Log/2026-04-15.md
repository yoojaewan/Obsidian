## 확정된 설계 결정

### 파츠 해상도
- 모든 파츠(초상화 + 스프라이트) 512×512 통일
- 스프라이트 화면 표시 크기는 줌 컨트롤로 사용자에게 맡김 (고정값 없음)
- 보스 스프라이트는 1024×1024 별도 제작
- 권장 범위 제도 불필요 — 런타임 스케일로 대체

### 파츠 구조
- Eyes = 눈썹 포함 세트 (분리 없음)
- 스프라이트도 파츠 조합 방식
- 스프라이트 바디는 통짜, 체형 차이는 scale로 처리
- 스프라이트 쿼터뷰 각도: 첫 바디 수작업 후 레퍼런스로 AI 생성

### Color System
- 용어 통일: 셰이더 적용 전 원본 이미지 = "그레이스케일 원본"
- 윤곽선은 순수 #000000, 셰이더에서 임계값(0.04) 이하는 검정 고정
- 그레이스케일 원본 중간톤 기준값: 128 (파츠 간 색상 일관성 위해 통일 필수)
- FaceBase / BodyBase 그라디언트 맵 적용 대상 추가 (skin_color uniform)
- 렌더링 순서: 그레이스케일 원본 → 그라디언트 맵 → 질감 오버레이 → 최종 출력

### 아트 스타일
- 원피스 애니메이션 스타일
- 셀 쉐이딩 — 그라데이션 없이 밝음/어두움 2단계로 딱 끊어서 표현
- 사실성보다 디자인 예술성 — 일치된 톤앤매너에서 오는 퀄리티
- 색상 절제 — 캐릭터당 2~3색

### 클래스 실루엣
| 클래스 | 실루엣 |
|---|---|
| 탱커 | 정사각형 — 사각 투구 + 넓은 어깨 |
| 파이터 (근딜) | 직사각형 — 풀아머 |
| 암살자 (근딜) | 웅크린 둥근형 — 스닉 자세 |
| 궁수 (원딜) | 세로 직선 + 캐릭터 키만한 대형 활 |
| 마법사 | 삼각형 — 뾰족하고 높은 모자 |

### 패션 컬러 공식
- 포인트 컬러 법칙 70:25:5 (일반/고급/레전더리 등급 적용)
- 클래스별 고유 컬러 톤 팔레트 (탱커 철회색, 마법사 보라/딥블루 등)
- 톤온톤/톤인톤/톤앤톤 공식 적용
- 소재 질감: metal / leather / cloth

### 장비 스키마
- 파츠 메타데이터에 `texture` 필드 추가 (장비 파츠에만 적용)
- 장비 레벨 스키마 별도 필요 (파츠 이미지 + main_color + 등급 + 이름)

### 앱 구조
- 하나의 앱 안에 캐릭터 탭 + 장비 탭
- 장비 탭: 장비/무기 제작
- 캐릭터 탭: 파츠 조합 + 장비 장착 + 초상화/스프라이트 미리보기
