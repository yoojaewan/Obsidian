# Art Concept Guide

## 목적

캐릭터/장비 아트 제작 시 일관성 유지를 위한 컨셉 가이드.
신규 캐릭터/장비 생성 범위 정의 시 기준으로 활용한다.

---

## 아트 스타일

- 원피스 애니메이션 스타일
- 굵고 명확한 윤곽선
- **셀 쉐이딩** — 그라데이션 없이 밝음/어두움 2단계로 딱 끊어서 표현
- 색상 절제 — 캐릭터당 2~3색
- 단순하고 임팩트 있는 실루엣
- 줌아웃 시 실루엣이 명확하게 읽혀야 한다

---

## 패션 컬러 공식

### 포인트 컬러 법칙 (70:25:5)

| 비율 | 역할 | 등급 적용 |
|---|---|---|
| 70% | 기본색 | 일반 장비 — 기본색만 |
| 25% | 서브색 | 고급 장비 — 서브색 추가 |
| 5% | 포인트색 | 레전더리 — 포인트색 강렬하게 |

### 등급별 컬러 정책

| 등급 | 컬러 방향 | 설명 |
|---|---|---|
| 일반 | 무채색 계열, 무난한 컬러 | 낡고 어두운 느낌 |
| 고급 | 채도 높고 쨍한 컬러 | 밝고 선명한 느낌 |
| 레전더리 | 클래스 톤의 가장 강렬한 버전 | 포인트색 극대화 |

### 클래스별 컬러 톤

클래스마다 고유 컬러 톤 팔레트를 가진다. 같은 클래스 장비끼리 맞춰 입으면 톤이 자연스럽게 맞아 완성도 있어 보인다.

| 클래스 | 컬러 톤 |
|---|---|
| 탱커 | 철회색/무채색 계열 |
| 파이터 | 적갈색/브라운 계열 |
| 암살자 | 다크그린/블랙 계열 |
| 궁수 | 카키/올리브 계열 |
| 마법사 | 보라/딥블루 계열 |

- 톤온톤: 같은 계열 색상, 밝기만 다르게
- 톤인톤: 같은 채도, 다른 색상
- 톤앤톤: 대비 톤으로 포인트

### 소재 질감

같은 톤이어도 소재 질감으로 입체감을 준다.

| 소재 | 클래스 |
|---|---|
| 철 | 탱커, 파이터 |
| 가죽 | 암살자, 궁수 |
| 천/로브 | 마법사 |

- 질감은 장비 메타데이터 JSON의 `texture` 필드로 정의
- 렌더링 순서: 그레이스케일 원본 → 그라디언트 맵 (색상) → 질감 오버레이 → 최종 출력

### 비율과 실루엣

- 탱커: 상체 크고 하체 작게 → 안정감, 묵직함
- 암살자/궁수: 상하 균형 → 날렵함
- 마법사: 세로로 긴 비율 → 위압감

## 클래스별 실루엣

줌아웃 상태에서도 클래스가 즉시 구분되도록 기하학적으로 다른 실루엣을 유지한다.

| 클래스 | 실루엣 형태 | 헬멧/장비 | 특징 |
|---|---|---|---|
| 탱커 | 정사각형 | 사각 투구 + 넓은 어깨 아머 | 옆으로 퍼지는 묵직한 형태 |
| 파이터 (근딜) | 직사각형 | 풀아머 | 세로로 긴 사각형 형태 |
| 암살자 (근딜) | 웅크린 둥근형 | 가죽 + 복면/후드 | 스닉 자세, 낮고 둥근 형태 |
| 궁수 (원딜) | 세로 직선 + 대형 활 | 가죽 | 캐릭터 키만한 장궁이 옆으로 튀어나옴 |
| 마법사 | 삼각형 | 뾰족하고 높은 모자 | 위로 솟는 형태 |

### 클래스 실루엣 제작 규칙

- 장비가 바뀌어도 클래스 실루엣 정체성이 유지되어야 한다
- 탱커 아머는 어깨가 넓게, 사각형 윤곽을 유지한다
- 파이터 아머는 세로로 긴 직사각형 윤곽을 유지한다
- 암살자는 스닉 자세 + 가죽 장비로 낮고 둥근 실루엣을 유지한다
- 궁수는 캐릭터 키만한 대형 장궁으로 실루엣 차별화한다
- 마법사 모자는 세로로 높게, 뾰족한 형태를 유지한다
