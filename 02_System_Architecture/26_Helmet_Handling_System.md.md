# Helmet Handling System

## Helmet Type
- OPEN
- HALF
- CLOSED

## 처리 원칙
- Helmet은 단순 이미지가 아니라 룰 오브젝트다
- Hair 레이어 hide / swap / keep 규칙을 가진다

## 예시
OPEN
- Hair 유지

HALF
- Hair_Side 일부 숨김

CLOSED
- Hair_Back 숨김
- Hair_Side 일부 숨김
- 필요시 Hair_Front helmetfit 버전으로 swap

## 주의
- 모든 Hair x Helmet 개별 대응은 금지
- Hair category/tag 기반 규칙 처리 우선
