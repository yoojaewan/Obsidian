# UI Layout — Asset Generator

## 환경

- 타깃 해상도: 1920 × 1080 (FHD)
- 프레임워크: Godot (예정)
- 원본 이미지 크기: 512 × 512
- UI 표시 크기: 384 × 384 (75% 스케일)

---

│                    │                                                │

│      420px         │                 1500px                         │

│  │              │  │                                                │

│ ─────────────────  │                                                │

## 미리보기 패널 상세 (420 × 1040)

두 칸 모두 동일한 구조.

```
┌─────────────────────┐  ← 섹션 상단 (y=0)
│  패딩 20px           │
│  타이틀 "초상화"  30px │
│  패딩 없음            │
│ ┌─────────────────┐  │
│ │                 │  │
│ │   이미지 영역    │  │  384 × 384px
│ │   384 × 384     │  │
│ │                 │  │
│ └─────────────────┘  │
│  하단 컨트롤   86px   │  ← 버튼/줌 등
└─────────────────────┘  ← 섹션 하단 (y=520)
```

### 하단 컨트롤 영역 (각 섹션 86px)

| 섹션 | 컨트롤 내용 |
|---|---|
| 초상화 | [PNG 내보내기] 버튼 |
| 스프라이트 | [PNG 내보내기] 버튼 + 줌 슬라이더 |

> 스프라이트 줌은 런타임 스케일이므로 표시 전용. 원본 해상도에 영향 없음.

## 컨트롤 패널 — 장비탭 (1500 × 1040)

```
┌──────────────────────────────────────────────────────────────────────┐
│  장비 종류:  ○ Armor   ○ Helmet   ○ Weapon              ← 라디오버튼  │  40px
├───────────────────────────────────┬──────────────────────────────────┤
│                                   │                                  │
│   파츠 그리드                      │   속성 패널                       │
│   (그레이스케일 원본 썸네일)         │                                  │
│   ~850px                          │   ~650px                         │
│                                   │                                  │
│   [▢][▢][▢][▢]                   │   main_color   [████████]        │
│   [▢][▢][▢][▢]                   │                                  │
│   [▢][▢][▢][▢]                   │   inner_color 프리셋              │
│   [▢][▢][▢][▢]                   │   [●][●][●][●][●]               │
│   ...  (스크롤)                    │                                  │
│                                   │   등급                           │
│                                   │   ○ 일반  ○ 고급  ○ 레전더리      │
│                                   │                                  │
│                                   │   이름                           │
│                                   │   [____________________________] │
│                                   │                                  │
│                                   │   [저장]        [초기화]          │
└───────────────────────────────────┴──────────────────────────────────┘
```

### 구역 수치

| 구역 | 너비 | 높이 | 비고 |
|---|---|---|---|
| 종류 선택 (라디오) | 1500px | 40px | 상단 고정 |
| 파츠 그리드 | 850px | 1000px | 세로 스크롤 |
| 속성 패널 | 650px | 1000px | 고정 |

### 파츠 그리드 썸네일

- 썸네일 크기: 192 × 192px (원본 512의 37.5%)
- 한 줄 4개: 4 × 192 + 패딩 = ~850px
- 선택 시 하이라이트 테두리

### 속성 패널 상세

| 항목 | 컨트롤 | 비고 |
|---|---|---|
| main_color | 컬러 피커 | 그라디언트 맵 적용 기준색 |
| inner_color 프리셋 | 색상 버튼 5개 | 헤어 컬러 계열 샘플, 클릭 시 미리보기 반영 |
| 등급 | 라디오 버튼 | 일반 / 고급 / 레전더리 |
| 이름 | 텍스트 입력 | |
| 저장 | 버튼 | 장비 아이템 데이터로 저장 |
| 초기화 | 버튼 | 현재 설정 리셋 |

## 탭별 미리보기 패널 콘텐츠

| | 위 칸 | 아래 칸 |
|---|---|---|
| 장비탭 | 장비 초상화 (단독 렌더) | 스프라이트에 장착된 모습 |
| 캐릭터탭 | 캐릭터 초상화 | 캐릭터 스프라이트 |
